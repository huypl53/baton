# docs/ — Index

Canonical wiki for all documentation in this repository.

| File | Purpose | When to use |
|------|---------|-------------|
| `project-overview-pdr.md` | Problem statement, solution summary, goals, scope | Onboarding; understanding why the repo exists |
| `codebase-summary.md` | File-by-file reference table for every path in the repo | Locating a specific file quickly |
| `code-standards.md` | Bash conventions, plugin/skill authoring rules, alias patterns | Before editing any script or adding a new skill |
| `system-architecture.md` | Component diagram, setup.sh flow, plugin loading, encryption layer, statusline data flow | Understanding how everything connects |
| `project-roadmap.md` | Planned features grouped by milestone | Deciding what to work on next |
| `catalyst-plugin.md` | Catalyst meta-plugin documentation | Building project-specific workflow plugins |

## Note

This repo contains **plugins only**. Personal configurations (settings.json, CLAUDE.md, shell aliases) are managed separately.
